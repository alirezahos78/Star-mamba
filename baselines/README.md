# Baseline Artifacts

This folder contains lightweight artifacts needed to document the external comparison baselines used for Star-Mamba.

Included:

- local training scripts used to run each baseline,
- local training logs with final reported accuracies,
- no external model source trees,
- no datasets,
- no checkpoints.

The full upstream projects should be cited and fetched separately:

- EfficientVMamba: `https://github.com/TerryPei/EfficientVMamba`
- Vim: `https://github.com/hustvl/Vim.git`

See `../BASELINES.md` for commit hashes, result tables, and reproducibility notes.
